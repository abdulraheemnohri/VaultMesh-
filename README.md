# VaultMesh Flutter X

A secure, offline Flutter app for managing sensitive data like passwords, notes, API keys, and documents.

## Features
- AES-256 encryption
- Master password authentication
- Biometric unlock
- Encrypted backup system
- Modern Material 3 UI

## Getting Started

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Dependencies
- Flutter 3.0+
- Hive for local storage
- Encrypt for AES-256 encryption
- Local Auth for biometric authentication
